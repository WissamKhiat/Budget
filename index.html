    📱 Installer l'app sur téléphone

    Pour recevoir les vraies notifications push, l'app doit être hébergée en ligne (GitHub Pages, Netlify, etc.) et installée sur votre téléphone.

    📋 Étapes pour installer :

    1. Hébergez les 3 fichiers sur GitHub Pages (gratuit)
    2. Ouvrez le lien sur votre téléphone
    3. iPhone : Partager → "Sur l'écran d'accueil"
    4. Android : Menu → "Ajouter à l'écran d'accueil"
    5. Activez les notifications dans l'onglet Alertes
